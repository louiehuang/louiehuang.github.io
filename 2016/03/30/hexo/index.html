github + hexo 搭建博客

git仓库必须为账号的用户名，否则pages不会分配空间
参见说明：You can only use your own account name for a User or Organization Page repository. A repository like joe/bob.github.io will not build a User Pages site.
https://help.github.com/articles/user-organization-and-project-pages/

验证git连接：
ssh -T git@github.com
出现提示则输入 yes

在本地新建文件夹，命名与仓库名相同，louiehuang.github.io

初始化：sudo hexo init

生成静态页面：sudo hexo g

本地运行：sudo hexo s
可在 http://localhost:4000 查看效果

然后执行命令：

npm install hexo-deployer-git --save

sudo hexo deploy
