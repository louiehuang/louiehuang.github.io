09 Fibonacci and Jump Floor

Problem Statement:

1 计算斐波那契数组

2 跳台阶，一次可以跳1级或2级，求跳上n级台阶有多少种跳法

3 跳台阶，一次可以跳1到n级，求跳上n级台阶有多少种跳法

4 我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？


Problem Statement:

1 Calculate Fibonacci Number

2 Jump floor, you can jump 1 or 2 steps each time. How many ways to jump from floor 0 to floor n?

3 Jump floor, you can jump 1 to n steps each time. How many ways to jump from floor 0 to floor n?

Question4:

For a matrix with length n, consider how we cover its left-most area.

(1) If we put a small matrix vertivally (cover 1*2), then we need to fill the sub matrix of length n-1.

(2) If we put a small matrix horizontally (cover 2*1), then we need to fill the sub matrix of length n-2 and we can put another 2*1 small matrix below the one we’ve put.

So this is still essntially a Fibonacci problem.
