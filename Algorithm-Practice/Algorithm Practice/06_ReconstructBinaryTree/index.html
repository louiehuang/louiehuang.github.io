06 Reconstruct a Binary Tree

Problem Statement:

输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。


Problem Statement:

Given the pre-order traversal and in-order traversal of a binary tree, build this tree. You may assume there will be no duplicate in either pre-order or in-order list.


Solution:

We can use pre-order list to determine the root and in-order list to determine this root’s left and right subtrees.

Suppose pre-order list is {5,4,2,7,6,8}, and in-order list is {2,4,5,6,7,8}. Now consider the 1st element in preorder list, we can know that 5 is the root, and {2,4} are in its left subtree while {6,7,8} are in its right subtree.

For 5’s left subtree, just consider the element next to 5 in the preorder list, which is 4. So 4 is the root of 5’s left subtree, check inorder list, we can infer {2} is in its left subtree and 4 has no right child.

For 5’s right subtree, we need to do somework - determine the size of the left subtree to get the correct index or the root of the right subtree. $|{2,4}| = 2$, so the root should be at index 0 + 2 + 1 = 3, which is 7, where 0 is the index of 5, 2 is the size of the left subtree.

Repeat this process, we will build the tree.
