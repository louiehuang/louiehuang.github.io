03 Search Two Dimensional Array

Problem Statement:

在一个二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。


Problem Statement:

This problem is identical to LC-74:

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

  • Integers in each row are sorted from left to right.
  • The first integer of each row is greater than the last integer of the previous row.

Given target = 3, return true.


Solution:

We can start from top-right corner, from the problem statement we know that all the elements that are left to it are smaller than it and all the elements that are down to it are greater than it. So we simply compare this element with our target and move left or down.
