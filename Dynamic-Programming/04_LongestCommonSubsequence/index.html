04 Longest Common Subsequence

Description

Given two sequences, find the length of longest subsequence present in both of them. Both the strings are of uppercase.

Examples:
LCS for input Sequences “ABCDGH” and “AEDFHR” is “ADH” of length 3.
LCS for input Sequences “AGGTAB” and “GXTXAYB” is “GTAB” of length 4.

Input:
First line of the input contains number of test cases T, the T test cases follow.
Each test case consist of 2 space separated integers A and B denoting the size of string str1 and str2 respectively. The next two lines contains the 2 string str1 and str2 .

Output:
For each test case print the length of longest common subsequence of the two strings .

Solution (find length)

We have two strings, assume the length of the first string is m, and the length of the second string is n. We have two variables, it natural to think use both of them. Let’s try so. Use dp[i][j] to denote the length of the longest subsequence of str1.substring(0,i) and str2.substring(0,j).

  1. If str1[i-1] == str2[j-1](last characters of two strings), then dp[i][j] = dp[i-1][j-1] + 1. For example, two substrings are ABCDGH and AEDFH, we just need to find the longest subsequence of ABCDG and AEDF.
  2. If str1[i-1] != str2[j-1], then dp[i][j] = max{dp[i-1][j], dp[i][j-1]}.

Extension (find subsequence)

The solution above could give us the length of the longest subsequence. But how do we find the actual subsequence?

Let’s see the table that above solution will produce for input str1 = "AGGTAB", str2 = "GXTXAYB". And the longest common sequence is GTAB.

We can start from the bottom-right corner dp[m][n]. Fordp[6][7], str1[5] == str2[6] == 'B', so we know that B is in the common subsequence and we need to check dp[m-1][n-1](diagonally) next time. For dp[5][6], str1[5] == 'A' != 'Y' == str2[6], we will check left cell and top cell, since dp[5][5] = 3 > 2 = dp[4][6], we know the result of dp[5][6] = 3 came from dp[5][5], so we go left. Then produce A, go diagonally; do not output, go left, produce T, go diagonally; do not output, go top or left; produce G, stop.

In this manner, we will have BATG, reverse it will be our final answer GTAB
