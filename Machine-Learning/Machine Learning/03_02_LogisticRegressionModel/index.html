If our correct answer y is 0, then the cost function will be 0 if our hypothesis function also outputs 0. If our hypothesis approaches 1, then the cost function will approach infinity. (more penalty)

If our correct answer y is 1, then the cost function will be`0 if our hypothesis function outputs1. If our hypothesis approaches0`, then the cost function will approach infinity.

Note that writing the cost function in this way guarantees that $J(θ)$ is convex for logistic regression.
